Game

House-keeping
Game.title:String
Sets the page header.
Game.title = "my game";
✎
Game.homepage:String
Set a homepage link to appear in the standalone player.
//you will only be able to see this in the stand-alone player
Game.homepage = "http://www.warpdoor.com";
✎
Game.background:Int
Sets color of the background in the standalone player.
Game.background = 0xff0000;
✎
Game.foreground:Int
Sets color of title text + links in standalone player.
Game.foreground=0x00ff00;
✎
Game.prompt(description,defaultText):String
trace(Game.prompt("enter a string","default value"));
✎
Game.save(key:String,value:String)
Save some data permanently that will persevere even if the game is closed and reopened.
Game.save("myVariable","32");
trace(Game.load("myVariable"));
✎
Game.load(key:String):String
Load save game data stored with Game.save. Returns null if no key present.
Game.save("myVariable","32");
trace(Game.load("myVariable"));
✎
Game.editor():Bool
True when a game is in the editor, but false when being played in the player.
function update(){  
 Text.display(0,0,Game.editor());
}
✎
Game.time:Int
Counter. Incremented by one each frame (30 per second).
function update(){
 Text.display(0,0,Convert.tostring(Game.time));
}
✎
Game.restart()
Restarts the game, resetting all variables.
var i=0;
function update(){
 i++;
 Text.display(0,0,Convert.tostring(i));
  if (Input.justpressed(Key.R)){
    Game.restart();
  }
}
✎
