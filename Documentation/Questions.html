Why does this exist? Why another game engine?
Zeedonk's goal is to make it easy to make and share realtime games all within the browser.
Who made this?
At the time of launch Terry Cavanagh and Stephen Lavelle. A list of all post-launch contributions can be found here.

There are many technologies used in Zeedonk, the creators of which deserve full credit

  • CodeMirror
  • Haxe
  • OpenFl
  • hscript

The sound generator is based on sfxr.

(Also, in addition to using Haxe, some of the documentation for the standard haxe functions is taken from it basically verbatim, under the MIT license).

What programming language am I using here?
HScript, an interpreted dialect of Haxe.
What is Zeedonk's resolution?
240x150
Can I host games on my own website?
Yes! You'll be missing out on some fixes that we might push occasionally, but it's still a good idea to keep a backup.

  1. Go to the github repository
  2. Find the "Download ZIP" button and click it.
  3. Unzip it, and replace default.hx with your game's code (make sure it's utf-8)
  4. Put it up on a webserver (due to security concerns, it won't work on most browsers if opened in a browser directly from the file-system)
  5. going to play.html should now load your game

Is there any way to modify the contents of a shared game url?
If you want to share a link to a game you've made, but still be able to update it (in case there are bugs you want to fix, or new levels you want to add), here's how you do that:

  1. Click Share in the Zeedonk editor as you normally would.
  2. Copy the ID at the end of the shareable URL that Zeedonk just generated for you:

    http://www.zeedonk.net/play.html?p=6878337,

    and paste it at the end of this URL:

    https://gist.github.com/anonymous/6878337.

  3. If you're signed into GitHub, you should see a button labelled Fork. Press that. (If you don't have a GitHub account, you'll need to create one for the following to work).
  4. Now you have your own updatable Zeedonk file! Take the ID at the end of the URL of your fork (it will be different to the one you pasted above) and perform Step 2 in reverse to find out your shareable and updatable game URL.
  5. Now anytime you want to update it, just go back to the fork you made, make some changes, and the same game URL should be updated too!

Can I sell/distribute games made with Zeedonk?
You can do whatever you want with Zeedonk games basically (it's distributed under the MIT license - though your games don't need to be).
Zeedonk is too slow for my purposes. I want to turbo-charge it!
There are many optimization techniques possible - caching images being a common one. The example game SUPER SHOT is pretty heavily optimized.

If you want a turbocharged version of zeedonk, the engine that it's built on top of is also perfectly usable - it's much faster, and will be quite familiar - it takes a small bit of work to set up, however.

It's super cool, and called HAXEGON.
